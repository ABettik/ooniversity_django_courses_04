Information about PyBursa

First page made by Bohdan Volynskyi bohdan.volynskyi@gmail.com

PyBursa is one courses provided by ItBursa (itbursa.com) and located in Kharkiv. PyBursa course is about teaching people how to use Python programming languare and Django Web framework.

The course lasts for 2 months and gives a lot of practical knowledge in Python and Django. After user finishes this course he would be able to take a role of Junior Python Web developer.

Program

The course contains of offline/online lessons, video lectures and practical tasks. Each of task has its own deadline.

For more information about course program and the nearest course start you can find at itbursa.com
